## Upgrade Instructions

Follow these steps to update your documentation site to the latest version.

<Steps>
  <Step title="Backup Your Site" icon="database">
    Export your current configuration and content.

    ```bash
    npx dhananjay-docs export --backup ./my-backup
    ```
  </Step>
  <Step title="Install Update" icon="download">
    Pull the latest release using your package manager.
  </Step>
  <Step title="Platform-Specific Update" icon="terminal">
    Choose your platform below.
  </Step>
</Steps>

<Tabs>
  <Tab title="npm" icon="package">
    ```bash
    npm install dhananjay-docs@latest
    npx dhananjay-docs migrate
    ```
  </Tab>
  <Tab title="Yarn" icon="package">
    ```bash
    yarn add dhananjay-docs@latest
    npx dhananjay-docs migrate
    ```
  </Tab>
  <Tab title="Docker" icon="docker">
    ```bash
    docker pull dhananjaydocs/latest
    docker-compose up -d
    ```
  </Tab>
</Tabs>
